# Care Innovations Website

Premium corporate website for Care Innovations, a strategic business development and growth consulting firm founded in Bengaluru in 2001.

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Lucide Icons

## Local Development

```bash
pnpm install
pnpm dev
```

Open `http://127.0.0.1:3000`.

## Production Build

```bash
pnpm build
```
